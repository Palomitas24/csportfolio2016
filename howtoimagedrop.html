How To Do A Dropdown When You Click On An Image?

Ever wonder how did the text appeared and and then disappeared when you click on the image in the 'Which Sport is Better?' page? In this page I'm going to explain how I made this and explaining how the variables, fuctions, and everything else that works behind the coding.

The Whole Coding

This right here is the coding behind the Dropdown. I'll will show how to do it and how it works inside the coding. I'll explain it later, but for now I'm going to introduce you what I did.

What I have created?

What I created in the "Which Sport Is Better?' project, is when you click on the image (of which ever sport you wantet to learn from) the information of the sport pops down from the image. When you click on the image again, the information disappers and hides behind the image. I wanted to do this because it was a easy, but not simple, thing to do. What Is a Dropdown?

You are asking yourself, what is a dropdown? A dropdown is a list or items, or words, that appear when clicking on a button,text, or image. When you click on the image a dropdown will appear at the bottom of the image with information.

How Does the Image Dropdown Work?

To do the dropdown when you click on the image, you need two files: the HTML file and the Javascript file.

HTML File

In the HTML file, you have to start with putting your Id's. An id for you guys who don't know is the an is is an artibute specifies an unique id for a HTML element (the value must be unique within the HTML document). The id attribute is most used to point a style in a style sheet, and by Javascript (via the HTML document) to manipulate the element with the specific id. For the downdrop to work, you need an id for both the image you going to click on and the information you want to type about your topic so it will been shown at the bottom of the image when you click it and disapear when you click it again. For example, my id's for the dropdown was (I am going to include the soccer dropdown): for the image it was 'id="soccerpic"' and for the inforamtion about soccer, 'id="soccerInfo"'. This tells that in the Javascript file, where to find that id to do what part. Also on the information , I used 'display:"none"' because when the the page loads, I don't want the information to appear. You have to click on the image to see the information. Everytime you reload the page, the inforamation of the sport won't appear, until you click on the image. Even when you click on the image and the text appear at the bottom, when you refresh the page or go in again, the text (information) won't appear.You also need the format "div" to make what information you want to put inside the div. The div is the id for the dropdwon to appear and disappear when you click on the image.

Javascript File

Now lets talk about the Javascript file. As you see in the Images on the top of the page, there are many unifamiliar things you won't get. Lets talk about the first part of the code to do the dropdown. As you see, there is var, which is variables for short in programing. Variables declarations are processed before any code is execued. Variables can be thought of named containers. You can place data into these containers and then refer to the data simply naming the container. Once again I'm going to use the Soccer Dropdown as my example. As you see there are two variables for the soccer dropdown: 'var soccerInfo' and 'var 'soccerpic'. Now the other part of the code is the '= document.getElementbyId()'. This makes the id from the HTML file transfer to the Javascript file where variable in gets the id. For example my code was 'var soccerInfo = getElementbyId("soccerInfo"). So the soccer information is grabbed to do a function. I did the same thing for soccer image. We are now going to explain the middle and bottom of the code of the dropdown. Now you see the code 'soccerpic.addEventListener('click', function()' The addEventListener attach an event, for this instense clicking on the image, handler to an element, so that you can do something useful when an even is triggered. In this code we have an if else statement. This if else statement is when the webpage is open, the display will be none (for the text(information)) When you click on the image, the if statement says to turn the style of display fron "none" to "block" where the text is shown at the bottom of the image. Block makes the display to appear. The else statement is if "block" is shown and the image is clicked, display "none" again.
